# radha.exe | Radha Agarwal Portfolio

A premium, interactive developer portfolio showcasing a fusion of creative design, intelligent systems, and robust full-stack engineering. Built with the modern React ecosystem, this portfolio leverages highly interactive 3D rendering and fluid animations to create a deeply immersive digital experience.

## ✨ Key Features

- **Interactive 3D Tech Globe**: A stunning, draggable 3D geodesic sphere showcasing technical skills, built entirely from scratch using `Three.js` (WebGL + CSS3DRenderer).
- **Dynamic Social Dock**: A trigonometrically-calculated, animated quarter-circle radial menu built with `Framer Motion` for elegant social link interactions.
- **Glassmorphism UI & Neon Cursor**: A sleek dark mode design system featuring custom dynamic lighting effects, ambient glows, and a smooth tracking neon cursor.
- **Advanced Sticky Scrolling**: Highly custom scroll-linked layouts to showcase project previews dynamically alongside detailed project cards.
- **Fully Responsive**: Carefully engineered layouts to provide a seamless premium experience across desktop, tablet, and mobile devices.

## 🛠 Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS for complex animations
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [Three.js](https://threejs.org/)
- **Icons**: Devicon & Lucide React


## 📂 Project Structure

- `/app` - Next.js App Router pages, global layouts, and core CSS (`globals.css`).
- `/components` - Reusable UI components including the `TechGlobe`, `ConnectBubble`, `NeonCursor`, and `SkillList`.
- `/public` - Static assets, project showcase images, and fonts.
